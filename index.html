How to pull/push
  • 1. git clone (clone Repository) 2. git add * (fetch file to Repository) 3. git commit -m "" (commit message) 4. git push (push to main) note: git status (get status) 1. git branch (info) 2. git branch -name- (new branch) 3. git checkout -name- (change branch) 4. git merge -name- (merge branches)
